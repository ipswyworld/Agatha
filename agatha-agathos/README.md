# Agatha Agathos

## Role
Agatha Agathos is the reporting engine for Project Agatha. It is responsible for generating structured, verifiable 'Evidence Packages' following a detection event.

## Functionality
- Generates JSON-formatted evidence packages.
- Includes timestamps, target identifiers (IP/URL), cryptographic hashes of evidence, and violation categories.
- Ensures evidence integrity for downstream processing.
